# 🔍 Search Site - Busca de Imagens

Este é um site simples de busca de imagens que utiliza a API do [Unsplash](https://unsplash.com/developers). O usuário digita uma palavra e o site retorna imagens relacionadas.

## 🚀 Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- API do Unsplash
